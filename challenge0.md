      #Real World Scenarios better suited for Static and Dynamic Structures

      
0. An airplane has exactly 180 seats. Each seat may be empty or assigned to one passenger.
   Here I would use a static structure because it has fixed number of seats.
   The trade-off is that empty chairs still take up space.
1.A help desk receives an unpredictable number of support tickets throughout the day.
  I would use a dynamic structure because there are unpredictable number of tickets.
The trade-off is that there maybe less tickets someday.
2.A vending machine has 12 fixed slots for snack types.
  I would use a static structure because there are fixed number of sack types.
the trade-off is that the snack types maybe out.
3.A classroom has 30 numbered desks.
  I would use a static structure because there arefixed number of desk.
the trade off is that all 30 seat maynot even be filled.
